# App Icons

Place your base PNG here and auto-generate the required sizes.

Quick steps:
1. Save your image as `source.png` in this folder.
2. Install deps and generate icons:
	- `npm install` (in the client folder)
	- `npm run icons`
3. This produces:
	- icon-32.png (favicon)
	- icon-64.png (navbar/logo)
	- icon-180.png (apple-touch-icon)
	- icon-192.png (PWA)
	- icon-512.png (PWA)

Tip: Prefer transparent PNG for best results.
